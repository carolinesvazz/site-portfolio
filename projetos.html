Churraspy

Este projeto é um site que pode ser útil para uma seção específica de churrasco em um supermercado ou para cálculo de churrasco em um Delivery. Os dados dos alimentos e bebidas são oriundos de um webscrapping no site de um supermercado reginal (Supermercado Princesa em Maricá) e foram armazenados em um banco de dados local, o projeto está configurado e rodando no servidor do python anywhere. O projeto foi desenvolvido com colaboração de equipes, no qual participei da equipe de Back-End.

Linguagem utilizada: Python/SQL | Ferramenta utilizada: PyCharm
